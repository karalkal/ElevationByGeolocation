This little application will display the elevation of any location on planet Earth based on its geographicalcoordinates (Latitude and Longitude).
It derives its data from GeoNames geographical database using 4 of the most popular freely available methodologies.

API specs API description
Elevation - SRTM1

Webservice Type : REST

Url : api.geonames.org/srtm1?

Parameters : lat,lng;

request method: GET or POST

sample area: ca 30m x 30m

Result : a single number giving the elevation in meters according to srtm1, ocean areas have been masked as "no data" and have been assigned a value of -32768
